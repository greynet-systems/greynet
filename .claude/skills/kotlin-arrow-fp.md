# Kotlin + Arrow-kt ФП

## Принципы
- Ошибки — через `Either<Error, T>` / `Raise<Error>`, НЕ exceptions
- `either {}` + `.bind()` для композиции
- Чистые функции, минимум мутабельного состояния, допускается мутабельность в скоупе функции
- Side-effects изолированы на границе (main)
